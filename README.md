# pokedexcli
A Pokedex REPL

## Requirements
[Install go](https://go.dev/doc/install) in order to build the executable. I didn't include it in this project.

## Build
Run `go build && ./pokedexcli` at the root of the project

## Commands
- help
- exit
- catch `pokemon name`
- explore `area name`
- inspect `pokemon name`
- mapf
- mapb
- pokedex
